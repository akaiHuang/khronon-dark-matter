## Abstract

We derive the MOND interpolating function from the Crooks fluctuation
theorem applied to the Khronon DBI kinetic structure, and prove a
no-go theorem establishing that the pure scalar Khronon requires a
tensor field companion (Blanchet-Skordis 2025) to satisfy CMB constraints.
The resulting Khronon-Tensor theory predicts CMB identical to LCDM,
while producing MOND dynamics at galaxy scales, with the transition
acceleration a_0 derived to 0.3% accuracy.

## Key Results

- No-go theorem: pure Khronon cannot fit CMB + Omega_DM simultaneously
- Resolution: Khronon-Tensor (BS2025) gives c_s^2 = 0 exactly -- CMB = LCDM
- a_0 = 2 pi c^2 (1+Omega_b)/[r_d (1+z_dec)] = 1.197 x 10^{-10} m/s^2 (0.3% match)
- mu(x) = x/(1-e^{-sqrt(x)}) from Crooks fluctuation theorem
- Omega_DM = 0.268 from retrodictability extremal principle (1.1% match)
- Bullet Cluster: c_s^2 = 0 -- pressureless dust -- consistent
